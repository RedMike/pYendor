Class Application

source code

Main application class.

After creating, set floor and wall to the desired tiles, add your windows, set them as default, then call newLevel and showWindow to show your game window. Transit state to 0, preferably, then enter the update loop. Inside the update loop, while not app.exit, call app.update.
